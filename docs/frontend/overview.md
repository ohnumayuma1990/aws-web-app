# フロントエンドの概要

フロントエンドはReactを使用して構築されたシングルページアプリケーション（SPA）です。プレイヤーがブラウザを通じてゲームルームを作成、参加、およびチャットを行うためのインターフェースを提供します。

## 主要技術
- **React**: UIフレームワーク。
- **TypeScript**: 型定義の導入。
- **WebSocket API**: サーバーとのリアルタイム通信。

## セットアップ
フロントエンドのコードは `frontend/` ディレクトリにあります。
環境変数 `REACT_APP_WSS_URL` を通じて、接続先のWebSocket APIエンドポイントを指定します。
