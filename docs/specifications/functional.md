# 機能仕様書

本ドキュメントでは、オンライン対戦ゲームの主要な機能とルールについて記述します。

## 1. ルーム管理機能

### 1.1 ルーム作成
- ユーザーは新しいゲームルームを作成できます。
- 作成時に「プライベート」か「公開」かを選択できます。
- ルーム作成成功時、ランダムな6文字（英数字）のルームIDが発行されます。

### 1.2 ルーム検索
- ユーザーは「公開」設定のルームを検索し、一覧を表示できます。
- ルーム一覧にはルームIDと作成日時が表示されます。

### 1.3 ルーム参加
- ユーザーはルームIDを直接入力して参加できます。
- 公開ルーム一覧から選択して参加することも可能です。
- 参加時、ルーム内の他のユーザーに通知が送信されます。

### 1.4 ルーム退出
- ユーザーはいつでもルームから退出できます。
- 退出時、ルーム内の他のユーザーに通知が送信されます。
- ブラウザを閉じる、あるいは通信が切断された場合も自動的に退出処理が行われます。

## 2. ゲームプレイ機能

### 2.1 ゲーム開始
- ルーム内のユーザーはゲームを開始できます。
- ゲーム開始時、52枚のトランプデッキが生成され、シャッフルされます。

### 2.2 カードを引く
- 自分のターンに山札からカードを1枚引くことができます。
- 引いたカードの内容は、引いた本人にのみ通知されます。

### 2.3 カードを出す
- 自分のターンに手札からカードを場に出すことができます。
- カードを出すと、次のプレイヤーにターンが移ります。

### 2.4 アクション
- ユーザーは場に出されたカードに対してアクション（クリックなど）を行うことができます。
- 現在の実装では、アクションを行うことでスコアが10ポイント加算されます。

### 2.5 勝利条件
- スコアが100ポイントに達したプレイヤーが勝利となります。
- 勝利が決まるとゲームは終了状態（ended）となります。

### 2.6 再戦（リセット）
- ゲーム終了後、またはプレイ中にゲームを初期状態（waiting）に戻すことができます。
- スコアと手札はリセットされます。

## 3. コミュニケーション機能

### 3.1 チャット
- 同じルーム内のユーザー間でリアルタイムにテキストメッセージを送信できます。
- 送信したメッセージはルーム内の全ユーザーに即時表示されます。
